What data tells about data jobs

Which skills and technologies are in use. Which salary will you find in the market. A must read for data scientist, data analyst, business analyst, data engineer and database administrator.

Are you in a data carrear path? You love to extract, transform and load data? Understand what it is telling you about your business? Creating models on it? And are you not sure about what you should be learning or the technologies you should master? You are not alone. Fortunately Stack Overflow is here to help us again, actually its survey data from 2019.

In this article we will be answering the following questions:

  • Which skills and technologies each data job require?
    The goal with this question is to understand which Languages, Databases, Platforms, Web Framework and Dev Environment is used by each data job.
  • How is the distribution of salary in the top respondents countries?
    This question is to understand how the market is paying these jobs, and you understand your position in it.
  • How much is worth each skill/technology/education in dollar per hours?
    The goal is to make a predictor of salary based on your knowledge and skills.

To answer those, we will use the stackoverflow 2019 survey dataset. Thus all the conclusions below assume that this sample fairly represent the population.

Question 1: Which skills and technologies each data job require?

There are several jobs that works with data, but our interest lays on those that have the main objective to storage it or to analyse it. The data jobs we are interested in the survey are: ‘Data or business analyst’, ‘Data scientist or machine learning specialist’, ‘Database administrator’ and ‘Engineer, data’.

For each of these jobs we will explore: main Programming Language they use, the databases they work on, the platforms, Web Framework, the development environment, the operational system they work on and other technologies.

Follow below which skills you should know, and for those you don’t know yet, just click in the link next to it to learn a little about it.

Data or Business analyst

The first language of a data/business analyst, is the SQL (Structured Query Language), since it allows you to extract and manipulate data from databases. Following Python and R that are languages with several modeling/statistical functions and packages like Pandas.

SQL free courses: w3school, udacity, datacamp and codeacademy

For business analyst is common the use of windows, following some uses on a linux platform.

Data/Business analyst rarely has to work with a Web Framework.

If you are a business analyst, chances are you use a windows.

Most data/business analyst have a bachelor or a Master’s.

Data scientist or Machine learning specialist

Python is not an option if you are following this carrear. SQL and R are musts. 

Most servers will be linux based, so you should learn about it. They will be probably be hosted on AWS, so you should understand some of its services. You will come across Docker, since it will let you run a container of an application or you will use it to make an image of your application.

If you need to use a web framework, probably it will be to insert your model into production. Flask and Django can do the trick.

Since python is a must, pandas goes together. TensorFlow and Pytorch are there to create your complex model. Spark and Hadoop are needed to work on a heavily data environment.

Jupyter is a common dev environment for a data scientist. 

Doctoral and Master’s degree are dominant in this carrear, keep this in mind.

Data Engineer

Python is the most worked on language for this carrear, while classical languages like C and Java continues to have relevance in the field. SQL is still mandatory to work with databases.

Most servers will be linux based, so you should learn about it. They will be probably be hosted on AWS, so you should understand some of its services. You will come across Docker, since it will let you run a container of an application or you will use it to make an image of your application.

This carrear does not require a strong web framework knowledge.

Although your operational system can run a Windows or a MacOS, Linux knowledge is necessary for the servers. 

This carrear is dominated by bachelor or a Master’s degree.

Database Administrator

SQL, SQL and SQL. Any database administrator should understand SQL with great depth, including the language nuances of each kind of database.

The top 3 databases you will find are: Microsoft SQL server, Oracle and MySQL.

This carrear is dominated by bachelor degree, but have a representation with Master’s degree and incomplete university degree.

Question 2: How is the distribution of Salary in the top respondents 5 countries?

The datajobs salary distribution are showed in the graphs below, segmented by the top 5 respondents countries and Brazil. If you are working in any data job, you can check where you stand in this distribution. Take this as a guide to set goals and to do a health check on your job income.

If you are not in the countries below take caution to extrapolate from these. The discrepancy of United States against India is not uncommon on the dataset. United kingdom, Germany and France may have a similar distribution, but even for Europe it is still a strong assumption to extrapolate. If you want this information use the stackoverflow predictor, it has a different scope, but has fewer limitations.

Some observations of this dataset:

  • The salary was transformed to dollar per work hour to make a fair comparison.
  • It was clearly unfair to group countries together, due to the cost of living and the salary received in each country.
  • There was not enough data to make a further segment with data job type.
  • The other countries did not have enough data to show its distribution.

Question 3: How much is worth each skill/technology/education in dollar per hours?

The answer to this question is: it is not predicable by this variables, at least not by the way it was structured.

The goal was to train a simple model that would retrieve how much each skill contributes to the salary. The training features were restrained to describe technologies knowledge, experience and academic study. Examples: year of coding, formal education, job type, languages worked with, database worked with, platform worked with, web framework worked with, misc tech worked with, development environment worked with and operational system.

Most variables are binary, so they doesn’t include the depth of experience in it. Formal education does not include the prestige of the university/college, nor the grades.

Every model generated on those variables had a performance near to the benchmark, which was a simple median of target value.

The inability to predict may be explained by the simplicity of those variables, except that the years of code feature could not help to predict. This fact leads me to a hypothesis that others kinds of skills, abilities, network and even work methodologies have a stronger influence in a developer salary than its technical knowledge. Just to be very clear, this hypothesis is not validated in this post.

Some observations:

  • The dataset contain every kind of developers, not only data jobs.
  • The model was created by country, due the motives explained in the second question.
  • The first target value was ‘DollarPerHour’ and the second was the first logarithm.

Conclusions

The most important skills and technologies are listed for each data job.

There is a range of salary for data jobs depending on the country, to check the health of your income and to set goals.

The information collected by this survey on skill, experience, job type, technologies and education can’t predict your salary.
